conocé a tus diputados (meet your congressmen) - 2014

aplicación interactiva desarrollada para la honorable cámara de diputados de la nación argentina. el sistema permite visualizar datos de todos los diputados, incluyendo fotos, afiliación política, e información personal. imágenes y datos provistos por la cámara de diputados.

el proyecto incluyó la creación de herramientas personalizadas para generar la base de datos con la información de los diputados.

esta aplicación se instaló en una serie de pantallas táctiles dentro del edificio del congreso, y forma parte de una exhibición permanente.

diseño visual por paula castillo.

creado con processing.

software development

interactive application commissioned by the national congress of argentina. the system serves as a visualization of every congressman’s data, including photos, political affiliation, and some personal information. data and images were provided by the congress.

custom software tools were created to build the database with all the congressmen’s information.

this piece is permanently displayed in a series of touch screens installed in the congress building.

visual design by paula castillo.

made with processing.
